READ ME FIRST
=============

You have a little keep to explore. Play it like this:

    lux run world.lux

Type what you want to do — look, north, take key — and press enter. If you
get stuck, type help. Type quit when you want to stop.

Play it through once before you read any further. Go on.


...back? Good.

Here is the secret of this place: the keep you just walked through is
world.lux, and world.lux is a program you can read. Open it in your editor.
The rooms, the doors, the torch in the cellar — they are all written down in
there, in plain lux, and none of it is hidden from you.

That means you can change it. Try the smallest thing first: find a room's
description and rewrite it. Run it again. The keep changed because you changed
it.

When that feels easy, make the cellar lead somewhere new, or hide a third thing
to find. You are not playing the game anymore — you are building it.

    lux learn crawl     how a world is put together
    lux learn           everything else lux can do

Have fun. Break things. It's your keep.
